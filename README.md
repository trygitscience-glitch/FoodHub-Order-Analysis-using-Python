# FoodHub-Order-Analysis-using-Python
This project explores restaurant order data to uncover insights into customer behavior, delivery performance, and operational efficiency using Python.

Using Pandas, NumPy, Matplotlib, and Seaborn, I performed data cleaning, exploratory data analysis (EDA), visualization, and business insight generation to identify key trends affecting customer satisfaction and restaurant performance.

A key finding from the analysis showed how small changes in delivery time can significantly impact customer retention and ratings.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook
